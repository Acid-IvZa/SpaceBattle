# Space Battle

Космический аркадный vertical shooter в одном HTML-файле.

## Особенности
- Один HTML-файл без внешних библиотек
- Чистый JavaScript
- 6 типов врагов
- Волны и прогрессия сложности
- Пауза на ПК и телефоне
- Поддержка мобильного управления

## Запуск
Открой файл `SpaceBattle.html` в браузере.
